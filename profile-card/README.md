# Semantic Profile Card

## What I Built

A responsive semantic profile card with an image, name, role, bio, and portfolio link.

## CSS Concepts Practiced

* HTML structure
* CSS connection
* Selectors
* Box model
* Spacing
* Sizing
* Image control
* Display
* Typography
* Color
* Link states
* Responsive behavior
* Cascade and specificity
* CSS cleanup

## What I Learned

* I learned that `width: 100%` and `max-width` work together to make the card flexible without letting it grow too wide.
* I learned the importance of not relying on browser default styles. It is important to make intentional style choices so a webpage appears consistently across supported browsers.
* I learned that organizing a stylesheet makes it easier for another developer to understand which sections affect each part of the page.
* I learned the difference between cascade and specificity, and how both affect which CSS rules apply to elements on the page.
* I learned the importance of responsive testing. A web application should be usable on different screen sizes. Browser developer tools help mimic screen size changes so I can test responsiveness.
